Feed Filters

By default, a Feed will include all content. You apply filters to narrow down the content contained in a feed. You must specify at least one filter, and each filter must have at least one value.

Choose the filters you would like to apply by activating a filter from the list below. Once you activate a filter, you can specify the values for that filter. Filters that are not specified will be ignored.

Filters are always combined together by an "and" rule. As an example, if you select the "Getty Images Wholly Owned", a Byline Title of "Staff", and an Iptc Category of "Sports" only Staff images within the Getty Wholly Owned group which have an Iptc Category of Sports will be included in the Feed.

If you cannot define your feed using the filters below, work with the Ftp Feed Support team to create a Feed using the "Advanced Mode" feature.

Select any number of Sources. You can either include the selected sources, in which case only the selected sources will be included in the Feed; or exclude sources, in which case anything but the selected sources will be included in the Feed. As always, these are combined with other Filters, so if you are combining sources with Groupings, you need to ensure the Feed will work. For instance, an inclusion of WireImage with the GYI Wholly Owned Grouping will never include an asset, because an asset can never by GYI Wholly Owned and WireImage. To create a feed of GYI Wholly Owned or WireImage, use the Advanced Mode feature.
