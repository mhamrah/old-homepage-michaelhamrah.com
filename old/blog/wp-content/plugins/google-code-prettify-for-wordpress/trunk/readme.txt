== Description ==

This small wordpress plugin enable syntax highlighting of code snippets in your post using Google Code Prettify.

== Installation ==
1. Upload the plugin to your plugin directory (wp-content/plugins)
2. Activate the plugin through the 'Plugins' menu in WordPress

== Uninstallation ==
For uninstallation deactivate the plungin through the 'Plugins' menu in WordPress
